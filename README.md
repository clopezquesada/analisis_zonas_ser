Código correspondiente al Trabajo de Fin de Máster: "Análisis de las Zonas de Servicio de Estacionamiento Regulado de Madrid. Sistema Inteligente de Búsqueda de Aparcamiento.
" realizado por Carlota López Quesada para el Máster en Big Data y Business Analytics. En él se utilizan herramientas de Big Data, manipulación, modelado y visualización de datos para llevar a cabo un análisis de la posibilidad de encontrar aparcamiento en las zonas de estacionamiento de la ciudad de Madrid.
Se utilizan metodologías de Machine Learning para la predicción y agrupación de valores a partir de datos proporcionados por el Portal de Datos Abiertos del Ayuntamiento de Madrid. En concreto, se presta especial atención al uso de datos de tráfico a tiempo real para que la recomendación de zonas de aparcamientos sea más adecuada. El trabajo se puede consultar en formato HTML descargando el archivo "TFM_Carlota_Lopez_Quesada.html" del siguiente enlace https://drive.google.com/drive/folders/1fPBtNw-YFMmtE6yJXnV2WCN5jIJQaNzg?usp=drive_link. En esta carpeta de Drive se pueden consultar, además, los conjuntos de datos utilizados y la aplicación creada para aumnetar la experiencia de usuario.
